CS343: Databases and Web Centric Programming

Semester: 2nd Semester
Prerequisites: CS244

Course Description

This course introduces relational databases and their integration with modern web applications. Topics include the relational model and its mapping onto the object model, implementing database applications in web contexts, web services, server-side scalability, virtualisation, and cloud computing.

Learning Outcomes

  • Design and query relational databases using SQL
  • Map the relational model onto an object-oriented model
  • Implement database-backed web applications
  • Understand and use web services and APIs
  • Apply techniques for server-side scalability
  • Understand virtualisation and cloud computing concepts

Assessment

  • Practical Assignments (30%) — Database design and SQL tasks, plus full-stack web application project integrating a relational back-end with a web front-end
  • Project (20%) — Group project: design, implement, and deploy a database-backed web application, with written report and demo
  • Final Examination (50%) — Written examination covering relational theory, SQL, web services, and cloud concepts; minimum of 40% required to pass
