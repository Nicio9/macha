CS313: Computer Networks

Semester: 1st Semester
Prerequisites: CS144

Course Description

Introduction to networks in general and the internet in particular. Topics include network architecture and protocols, allocation of resources and congestion control, network security, and networked applications.

Learning Outcomes

  • Understand the layered architecture of computer networks and the internet
  • Describe common networking protocols and their roles
  • Analyse resource allocation and congestion control mechanisms
  • Understand fundamental network security principles and threats
  • Implement basic networked applications

Assessment

  • Practical Assignments (30%) — Programming assignments covering socket programming, protocol implementation, and network simulation using tools such as Wireshark and ns-3
  • Tests (20%) — Two written tests during the semester covering theoretical content
  • Final Examination (50%) — Comprehensive examination covering all course topics; minimum of 40% required to pass the course
