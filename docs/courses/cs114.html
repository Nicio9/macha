CS114: Introductory Computer Science 1

Semester: 1st Semester
Level: First Year
Credits: 12

Course Description

Introduction to basic computer programming; formulation and solution of problems by means of computer programming; data representation and variable types (including character strings, integers, floating point numbers and Boolean variables); assignment statements; conditional execution and iteration; static data structures (arrays and records); input and output (including graphics and sound); modular programming, recursion, testing and debugging; introduction to object-oriented programming.

Learning Outcomes

  • Understand fundamental programming concepts and their application
  • Formulate and solve problems using computer programming
  • Work with various data types and structures effectively
  • Design and implement modular programs
  • Understand and apply object-oriented programming principles
  • Debug and test programs systematically

Course Content

  • Basic programming concepts and problem-solving
  • Data representation and variable types
  • Control flow: conditional execution and iteration
  • Functions and modular programming
  • Arrays, records, and static data structures
  • Input and output operations (including graphics and sound)
  • Recursion and recursive algorithms
  • Introduction to object-oriented programming
  • Testing and debugging techniques

Assessment

Assessment is based on practical programming assignments, laboratory work, examinations, and projects that demonstrate mastery of the course concepts and practical programming skills.

This course includes comprehensive project specifications designed to give students hands-on experience with real-world programming challenges. For detailed project specifications and requirements, please visit the official course page on the SUN website.
