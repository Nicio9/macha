CS344: Program Design

Semester: 2nd Semester
Prerequisites: CS214

Course Description

This course covers principled approaches to designing medium-to-large software systems. Topics include program specifications as guidelines for program design, reusable frameworks for program design, testability of program designs, and the practical application of program design principles through the development of a medium-sized system.

Learning Outcomes

  • Write formal and informal specifications to guide program design
  • Apply reusable design frameworks and patterns
  • Design programs with testability as a first-class concern
  • Develop a medium-sized software system applying design principles in practice

Assessment

  • Design Assignments (25%) — Specification and design exercises requiring formal and informal documentation of software components
  • Software Project (35%) — Individual development of a medium-sized system applying course design principles; assessed on design quality, test coverage, and correctness
  • Final Examination (40%) — Written examination on design theory, patterns, and specification techniques; minimum of 40% required to pass
