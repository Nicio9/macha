CS412: Advanced Algorithms (712*)

Semester: 1st Semester
Prerequisites: CS214

Course Description

This course resumes the study of algorithms and data structures where it left off in CS214. Various algorithmic paradigms are covered, including divide-and-conquer, greedy algorithms, dynamic programming, and randomised algorithms. Algorithmic applications include graphs, heaps, and trees, as well as an introduction to the basics of the theory of computation.

Learning Outcomes

  • Apply divide-and-conquer, greedy, dynamic programming, and randomised algorithm paradigms
  • Analyse the correctness and complexity of advanced algorithms
  • Design and implement algorithms on graphs, heaps, and trees
  • Understand fundamental concepts in the theory of computation
  • Select appropriate algorithmic strategies for complex problem instances

Assessment

  • Algorithm Assignments (35%) — Problem sets requiring design, correctness proofs, and complexity analysis of algorithms across multiple paradigms
  • Tests (15%) — Two written tests during the semester
  • Final Examination (50%) — Comprehensive written examination; minimum of 40% required to pass
