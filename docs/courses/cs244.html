CS244: Computer Architecture

Semester: 2nd Semester
Credits: 16
Year: 2nd Year
Prerequisites: CS144

Course Description

This course provides a thorough understanding of computer architecture from the logic level up to the operating system interface. Students explore how hardware design decisions affect software performance and how programmers can write code that exploits modern processor features.

The course covers digital logic design, instruction set architectures, processor microarchitecture (pipelining, out-of-order execution, branch prediction), memory hierarchy, caching, and the basics of parallel computing. Students will work with a real instruction set architecture and develop an appreciation for the interaction between hardware and software.

Topics Covered

  • Boolean algebra and digital logic: gates, combinational and sequential circuits
  • Number representations: integers, floating-point (IEEE 754)
  • Instruction set architecture (ISA) design principles
  • Assembly language programming
  • CPU datapath and control unit design
  • Pipelining: hazards, forwarding, branch prediction
  • Out-of-order execution and superscalar processors
  • Memory hierarchy: SRAM, DRAM, caches, virtual memory
  • I/O systems and interrupt handling
  • Introduction to parallel architectures: multi-core, SIMD

Learning Outcomes

  • Describe and design basic digital circuits using Boolean algebra
  • Explain the key components of a modern CPU and their interactions
  • Read and write simple assembly language programs
  • Analyse the effect of pipelining and caching on program performance
  • Understand how the memory hierarchy affects data access patterns
  • Describe the basics of parallel processor design

Assessment

  • Practical labs and assignments (35%)
  • Written tests (25%)
  • Final examination (40%)

Recommended Textbook

Patterson, D. A., & Hennessy, J. L. (2020). Computer Organization and Design: RISC-V Edition (2nd ed.). Morgan Kaufmann.
