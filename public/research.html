Research Groups

The Computer Science Division hosts eight vibrant research groups working on cutting-edge problems across diverse domains including machine learning, software engineering, formal methods, and specialized computing applications. Our researchers collaborate with industry partners and international institutions to push the boundaries of computing science.

Theory and Applications of Automata and Grammars

Automata Theory Formal Languages Pattern Recognition
This group investigates nondeterministic finite automata, descriptional complexity, and applications of automata and grammars to practical problems. Research areas include pattern layout, image processing, music generation, and grammar correction. The group develops both theoretical foundations and practical applications of formal language theory.
Lead Researchers:
  • Brink van der Merwe (Professor, Head of Division)
  • Willem Bester (Junior Lecturer)
  • Lynette van Zijl (Professor)
Research Interests: Automata theory, formal languages, cellular automata, pattern recognition, music generation, grammar learning

Software Engineering and Verification

Formal Verification Program Analysis Model Checking
Founded in 1990, this group develops highly reliable system software using computer-aided verification, systematic testing, and defensive programming. The group has extensive experience in OS kernels, communication protocols, and the development of verification tools. Research focuses on ensuring correctness and security of critical software systems.
Lead Researchers:
  • Bernd Fischer (Professor)
  • Cornelia Inggs (Senior Lecturer)
  • Willem Visser (Part-time Professor)
Research Interests: Software engineering, formal verification, program analysis, model checking, symbolic execution, testing, software construction

Machine Learning and Artificial Intelligence

Machine Learning Optimization Swarm Intelligence
This active group addresses decision-making problems including sequential decision-making, planning, and search algorithms using probability and game theory. The group combines theoretical foundations with practical applications in big data analysis, earth observation, and radio interferometry. Research includes evolutionary computation, neural networks, and optimization techniques.
Lead Researchers:
  • Andries Engelbrecht (Professor)
  • Steve Kroon (Associate Professor)
  • Trienko Grobler (Lecturer)
  • Mkhuseli Ngxande (Lecturer)
Research Interests: Machine learning, optimization, swarm intelligence, evolutionary computation, hyper-heuristics, neural networks, data analytics, big data analysis, earth observation

Telkom-Siemens Centre of Excellence in ATM and Broadband Networks

Network Technologies Telecommunications Broadband Systems
This centre promotes research and development in broadband technologies and provides specialized training of postgraduate students in telecommunications expertise. The group works on advanced telecommunications technologies and network research with industrial partnerships.
Lead Researchers:
  • William (Bill) Tucker (Professor)
  • Willem Visser (Part-time Professor)
Research Interests: ATM (Asynchronous Transfer Mode), broadband networks, telecommunications, network protocols, internet technologies

Natural Language Processing and Semantic Computing

NLP Semantic Web Knowledge Representation
This group focuses on multilingual natural language processing, semantic computing, ontology development, and machine translation. Research includes semantic web technologies, knowledge representation and reasoning, and language generation techniques. The group works on making computers understand and generate human language across multiple languages.
Lead Researcher:
  • Laurette Pretorius (Associate Professor Extraordinary)
  • Maria Keet (Professor Extraordinary)
Research Interests: Multilingual NLP, semantic web, ontology development, semantic computing, machine translation, language generation, knowledge representation and reasoning

Computer Vision and Pattern Recognition

Computer Vision Pattern Recognition Bioinformatics
This group employs brain-inspired computing approaches to computer vision and pattern recognition. Research includes developing intelligent systems that can process visual information similar to biological systems, with applications in smart cameras, surveillance, and bioinformatics. The group combines deep learning with neuroscience-inspired algorithms.
Lead Researchers:
  • George Azzopardi (Associate Professor Extraordinary)
  • McElory Hoffmann (Senior Lecturer Extraordinary)
  • Mkhuseli Ngxande (Lecturer)
Research Interests: Computer vision, pattern recognition, brain-inspired computing, machine learning, smart cameras, image processing, surveillance systems, bioinformatics

Cognitive Robotics and Knowledge Representation

Robotics Knowledge Representation Automated Planning
This group covers logic-based knowledge representation, deterministic and probabilistic planning, and reinforcement learning for intelligent systems. Research includes developing robots that can reason about their environment and plan actions autonomously. The group integrates symbolic AI with modern machine learning approaches.
Lead Researcher:
  • Gavin Rens (Lecturer)
Research Interests: Cognitive robotics, knowledge representation and reasoning, automated planning, reinforcement learning, probabilistic reasoning, logic programming, uncertainty reasoning

Programming Languages and Software Quality

Programming Languages Software Testing Code Quality
This group investigates programming language design and implementation, software testing techniques including fuzzing, and code quality assurance. Research focuses on making programming more efficient, safer, and more reliable. The group contributes to both theoretical language design and practical tools for software developers.
Lead Researchers:
  • Willem Bester (Junior Lecturer)
  • Moeketsi Raselimo (Lecturer Extraordinary)
  • Fabian Yamaguchi (Professor Extraordinary)
Research Interests: Programming languages, software testing, fuzzing, formal methods, code analysis, program analysis, security, software construction
