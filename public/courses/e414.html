E414: Machine Learning

Semester: 1st Semester
Prerequisites: None

Course Description

This course covers statistical machine learning techniques with a focus on probabilistic modelling. Topics include dimension reduction techniques and machine learning methods based on maximum-likelihood, maximum-posterior, and expectation-maximisation estimates. Specific models covered include logistic regression, Gaussian mixtures, and hidden Markov models. This module is the Engineering Faculty equivalent of CS315.

Learning Outcomes

  • Apply dimension reduction techniques to high-dimensional data
  • Implement and evaluate maximum-likelihood and maximum-posterior estimators
  • Derive and apply the expectation-maximisation (EM) algorithm
  • Build and interpret logistic regression models
  • Understand and apply Gaussian mixture models for clustering
  • Model sequential data using hidden Markov models

Assessment

Assessment is based on practical assignments, examinations, and projects that demonstrate mastery of the course concepts.
