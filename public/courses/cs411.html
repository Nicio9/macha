CS411: Computer Networks (313*)

Semester: 1st Semester
Prerequisites: CS313

Course Description

This postgraduate course extends the undergraduate study of computer networks. Topics include advanced network architecture and protocols, resource allocation and congestion control, network security, and networked applications, studied at greater depth and with an emphasis on research-level understanding.

Learning Outcomes

  • Critically analyse network architecture and protocol design choices
  • Evaluate advanced resource allocation and congestion control strategies
  • Understand and assess modern network security techniques
  • Read and discuss current research in computer networking

Assessment

Assessment is based on practical assignments, examinations, and projects that demonstrate mastery of the course concepts.
