CS441: Machine Learning (741*)

Semester: 2nd Semester
Prerequisites: CS315

Course Description

This course covers a broad range of machine learning techniques and their theoretical foundations. Topics include similarity-based learning, information-based learning, error-based learning, kernel-based learning, reinforcement learning, unsupervised learning, ensemble learning, online learning, probabilistic learning, deep learning, and the evaluation of learning systems.

Instructor: Prof Andries Engelbrecht

Learning Outcomes

  • Understand and apply similarity-based learning methods
  • Understand and apply information-based learning methods
  • Understand and apply error-based learning methods
  • Understand and apply kernel-based learning methods
  • Understand and apply reinforcement learning techniques
  • Understand and apply unsupervised learning techniques
  • Understand and apply ensemble learning techniques
  • Understand and apply online learning methods
  • Understand and apply probabilistic learning models
  • Understand and apply deep learning architectures
  • Evaluate and compare machine learning models rigorously
  • Select appropriate learning algorithms for practical problems

Assessment

Assessment is based on practical assignments, examinations, and projects that demonstrate mastery of the course concepts.
