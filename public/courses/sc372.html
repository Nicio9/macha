SC372: Scientific Computing

Semester: 1st-2nd Semester
Prerequisites: None

Course Description

This course builds on SC272 to cover more advanced scientific computing topics. Topics include LaTeX for scientific document preparation, regular expressions, sorting and searching algorithms, random number generation, advanced Python with NumPy and SciPy, construction of data-processing pipelines, and Linux tools for scientific workflows.

Lecturer: Hanlie Hardoldt

Learning Outcomes

  • Produce professional scientific documents using LaTeX
  • Use regular expressions for text processing tasks
  • Implement and analyse sorting, searching, and random number algorithms
  • Apply advanced NumPy and SciPy techniques for numerical computing
  • Build data-processing pipelines for scientific analysis
  • Leverage Linux tools to automate and streamline scientific workflows

Assessment

Assessment is based on practical assignments and examinations that demonstrate mastery of advanced scientific computing concepts.
