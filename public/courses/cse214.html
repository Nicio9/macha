CSE214: Object-Oriented Programming

Semester: 1st Semester
Prerequisites: None

Course Description

This course introduces problem formulation and solution by means of computer programming in an object-oriented setting. Topics include principles of testing and debugging, key concepts in object orientation (abstraction, encapsulation, inheritance, and polymorphism), design patterns, searching and sorting algorithms, complexity theory, and dynamic data structures.

Learning Outcomes

  • Formulate and solve problems using object-oriented programming
  • Apply testing and debugging techniques
  • Understand and apply core OO concepts: abstraction, encapsulation, inheritance, and polymorphism
  • Use common design patterns in program design
  • Implement and analyse searching and sorting algorithms
  • Understand basic complexity theory and algorithmic efficiency
  • Implement dynamic data structures such as linked lists and trees

Assessment

  • Programming Assignments (40%) — Regular coding assignments implementing OO concepts, data structures, and algorithms in Java; includes automated testing via a marking system
  • Tests (10%) — One written class test during the semester
  • Final Examination (50%) — Written examination covering OO theory, design patterns, and algorithmic analysis; minimum of 40% required to pass
