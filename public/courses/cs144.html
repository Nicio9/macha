CS144: Introductory Computer Science 2

Semester: 2nd Semester
Level: First Year
Prerequisites: CS114

Course Description

Further formulation and solution of problems by means of computer programming; introductory data structures and algorithms in an object-oriented set-up; key concepts in object orientation: inheritance and polymorphism; design patterns as abstractions for the creation of reusable object-oriented designs; searching and sorting algorithms; complexity theory for the analysis of algorithms; fundamental methods in the design of algorithms; dynamic data structures; regular expressions and finite automata.

Learning Outcomes

  • Understand and apply object-oriented programming principles
  • Implement and analyze data structures and algorithms
  • Design efficient algorithms using various design patterns
  • Understand complexity theory and big-O notation
  • Apply searching and sorting algorithms effectively

Course Content

  • Object-oriented programming concepts (inheritance, polymorphism, encapsulation)
  • Design patterns and their applications
  • Data structures: arrays, lists, stacks, queues, trees
  • Sorting and searching algorithms
  • Algorithm analysis and complexity theory
  • Regular expressions and finite automata

Assessment

Assessment is based on practical programming assignments, examinations, and projects that demonstrate mastery of the course concepts and practical programming skills.
