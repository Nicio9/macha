CS495: Functional Programming (795*)

Semester: Not presented Semester
Prerequisites: None

Course Description

This course explores functional programming as a style of programming that is fundamentally different from imperative programming. Functional programs tend to be shorter, easier to reason about, and more amenable to formal verification. The course covers the theoretical foundations of functional programming and practical applications using Haskell.

Recommended reading: Programming in Haskell by Graham Hutton; Learn You a Haskell for Great Good! (available free online).

Learning Outcomes

  • Write programs in a purely functional style using Haskell
  • Understand and apply core functional concepts: higher-order functions, lazy evaluation, and type classes
  • Reason formally about functional programs
  • Appreciate the advantages of functional programming for correctness and maintainability

Assessment

  • Project 1: 25%
  • Project 2: 25%
  • Presentation: 25%
  • Practical exam: 25%
