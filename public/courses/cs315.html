CS315: Machine Learning

Semester: 1st Semester
Prerequisites: CS144

Course Description

This course covers statistical machine learning techniques with a focus on probabilistic modelling. Topics include dimension reduction techniques and machine learning methods based on maximum-likelihood, maximum-posterior, and expectation-maximisation estimates. Specific models covered include logistic regression, Gaussian mixtures, and hidden Markov models.

Learning Outcomes

  • Apply dimension reduction techniques to high-dimensional data
  • Implement and evaluate maximum-likelihood and maximum-posterior estimators
  • Derive and apply the expectation-maximisation (EM) algorithm
  • Build and interpret logistic regression models
  • Understand and apply Gaussian mixture models for clustering
  • Model sequential data using hidden Markov models

Assessment

  • Practical Assignments (35%) — Programming assignments in Python implementing and evaluating machine learning models on real datasets
  • Tests (15%) — Two written tests during the semester covering probabilistic theory and model derivations
  • Final Examination (50%) — Comprehensive written examination; minimum of 40% required to pass the course
