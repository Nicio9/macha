CS214: Data Structures and Algorithms

Semester: 1st Semester
Credits: 16
Year: 2nd Year
Prerequisites: CS114

Course Description

This course provides an in-depth study of fundamental data structures and the algorithms that operate on them. Students will analyse the efficiency of algorithms and data structures using Big-O notation, and learn how to select the appropriate structure for a given problem. Topics include arrays, linked lists, stacks, queues, trees, heaps, hash tables, and graphs, as well as algorithms for sorting, searching, and graph traversal.

A strong emphasis is placed on both theoretical analysis and practical implementation. Students will implement data structures and algorithms from scratch and apply them to solve real-world computational problems.

Topics Covered

  • Algorithm analysis and asymptotic complexity (Big-O, Ω, Θ)
  • Arrays, linked lists, stacks, and queues
  • Trees: binary trees, binary search trees, AVL trees, B-trees
  • Priority queues and heaps
  • Hash tables and collision resolution strategies
  • Graphs: representations, BFS, DFS, shortest paths (Dijkstra, Bellman-Ford)
  • Minimum spanning trees (Prim, Kruskal)
  • Sorting algorithms: mergesort, quicksort, heapsort, radix sort
  • Dynamic programming
  • Greedy algorithms

Learning Outcomes

  • Analyse the time and space complexity of algorithms using asymptotic notation
  • Implement and use core data structures: lists, trees, graphs, and hash tables
  • Apply standard graph algorithms to solve connectivity and shortest-path problems
  • Design solutions using dynamic programming and greedy strategies
  • Select and justify appropriate data structures for specific problems
  • Implement efficient sorting and searching algorithms

Assessment

  • Practical programming assignments (40%)
  • Written tests (20%)
  • Final examination (40%)

Recommended Textbook

Cormen, T. H., Leiserson, C. E., Rivest, R. L., & Stein, C. (2022). Introduction to Algorithms (4th ed.). MIT Press.
