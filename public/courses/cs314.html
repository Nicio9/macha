CS314: Concurrency

Semester: 1st Semester
Prerequisites: CS144

Course Description

This course covers the principles of concurrent systems and process management in operating systems. Major topics include message passing programming, shared memory programming, parallel computer architecture, process management in operating systems, performance analysis, and parallel programming models and design.

Textbook: An Introduction to Parallel Programming, Peter Pacheco, Morgan Kaufmann.

Instructor: Dr Cornelia Inggs

Learning Outcomes

  • Understand different parallel programming models and environments
  • Use and choose among various shared-memory and distributed-memory APIs
  • Apply decomposition, ordering, and mapping techniques to identify concurrency
  • Organise tasks and data to choose the correct algorithm structure and supporting structures for implementation
  • Perform performance analysis including efficiency, speedup, overheads, and scalability
  • Understand the operating system's responsibility for managing processes, including creation, scheduling, and synchronisation

Assessment

Assessment is based on practical assignments, examinations, and projects that demonstrate mastery of the course concepts.
