SC272: Scientific Computing

Semester: 1st-2nd Semester
Prerequisites: None

Course Description

This course introduces students to scientific computing using the Linux command line and Python. Topics include variables, types, control structures, functions, strings, unit testing, and basic data processing in Python, as well as numerical computing with NumPy, data visualisation with matplotlib, and curve fitting techniques.

Lecturer: Indren Govender

Learning Outcomes

  • Use the Linux command line for scientific workflows
  • Write Python programs using core language features and best practices
  • Apply unit testing to verify program correctness
  • Use NumPy for numerical data processing
  • Create visualisations using matplotlib
  • Perform curve fitting and interpret results

Assessment

Assessment is based on practical assignments and examinations that demonstrate mastery of scientific computing concepts.
